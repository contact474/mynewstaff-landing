## STEP 2 — Qualification Call (15 min, optional for General tier)

**Tier rules:**
- **General ($1K):** Apply + pay. No call required. Fast lane.
- **VIP ($2.5K):** 15-min call with Luka OR Los. Confirm fit + dietary for dinner.
- **Deploy ($5K):** 30-min call with Luka. Scope the done-for-you. Confirm stack compatibility.

**Call script (short):**
1. "Tell me about the business in 60 seconds."
2. "If the intensive works perfectly, what does Monday morning after look like?"
3. "What's your #1 blocker — leads, conversion, delivery, or content?"
4. "Any reason NOT to do this that I should know about?"
5. "Cool. Here's the link. Payment locks your seat — we're at [X/20]."

**Payment:** Stripe link via GHL. Wire or crypto for Deploy tier.

## STEP 4 — Pre-Event Prep Required from Attendee

Send 7 days out via email + Signal:

1. **Accounts to create (if they don't have):**
   - Claude Pro/Max ($20-100/mo) — we'll cover first month for Deploy tier
   - GitHub account
   - Vercel account
   - Cloudflare account
   - Anthropic API key (free tier OK to start)
   - OpenAI API key (optional)

2. **Laptop requirements:**
   - Mac OR Windows, 16GB RAM recommended
   - Admin access (must be able to install)
   - Charger, mouse optional
   - Node 20+, Git installed

3. **Business info to bring:**
   - Brand voice doc or 3 sample posts
   - Top 10 customer pain points
   - Current lead sources and CPL
   - Current content schedule
   - Access to their website / domain / DNS

4. **Mindset:**
   - Clear 3 days — no Zoom calls, no client emergencies planned
   - Come with ONE outcome in mind, not ten
   - Ready to ship messy over perfect
